## CRUD API Rest Spring Boot 3 com Design Patterns

CRUD de uma API Rest de clientes com endereço usando o Spring Boot 3 e API ViaCep, com armazenamento em banco em memória H2 Database. E documentação via Swagger.

---

### Tecnologias utilizadas

- Spring Boot 3
- Spring Data JPA
- Spring Web
- H2 Database
- Java 11
- Maven
- Swagger
- API ViaCep
